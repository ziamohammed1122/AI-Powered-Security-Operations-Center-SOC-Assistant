# AI-Powered-Security-Operations-Center-SOC-Assistant
📌 Description

A real-time cybersecurity dashboard that displays security alerts, threat scores, and vulnerability data in a unified interface. It provides live insights into system security using interactive visualizations and real-time data integration.

🚀 Features
Real-time alert monitoring (total, critical, incidents)
Threat score and severity visualization
Interactive charts using Canvas 2D (gauges, donuts, bars, radar)
24-hour alert timeline tracking
Live CVE data from NIST NVD & CISA APIs
CVSS-based vulnerability analysis
Responsive dark-themed UI with glassmorphism design
🛠️ Tech Stack
HTML5
CSS3
JavaScript
Canvas 2D API
REST APIs
⚙️ How It Works

The dashboard runs in a web browser and fetches real-time data from APIs and monitoring sources. JavaScript processes the data and updates the UI dynamically using charts and alert panels.

👥 User Roles
Admin – Manages system settings and users
Security Analyst – Monitors and analyzes alerts
SOC Analyst – Responds to incidents
Manager – Views reports and insights
📊 Data Sources
System & network monitoring tools
Endpoint devices and servers
External APIs (NIST NVD, CISA)
📌 Conclusion

This project provides a centralized platform for monitoring and analyzing cybersecurity threats in real time, helping organizations respond quickly and effectively.
